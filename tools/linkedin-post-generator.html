How to get a free API key

  1. Go to console.groq.com/keys
  2. Sign up or log in (free)
  3. Click "Create API Key"
  4. Copy the key and paste it above

Your API key is stored only in your browser's local storage. It is never sent to any server other than Groq's API. Groq's free tier allows 30 requests/minute.
